This project not only underscores my adeptness in crafting mobile solutions but also highlights my proficiency in working with databases to ensure seamless appointment management. In addition to robust database integration, the app showcases my skills in designing intuitive and user-friendly interfaces.

LegalHub App Release Notes - Version 1.0.0

New Features:

1. User Authentication: • Firebase authentication is implemented for secure and seamless user authentication. • Lawyer validation process during sign-up ensures the correct assignment of user roles.
2. Appointment Management: • Clients can schedule and track appointments easily within the app. • Lawyers have a dedicated schedule displaying all appointments booked with them. • Lawyers can reschedule appointments, providing flexibility for both users and legal professionals.
3. Service Information: • Users can explore and view all the services offered by a law firm through the app. • Lawyers have the ability to add new services to the law firm's offerings.
4. Role-Based Access: • User roles are assigned based on the password used during sign-up. • If the password is 'admin1', the user is assigned the role of a lawyer. • Lawyers can utilize additional functionalities within the app, enhancing their experience.
5. Password Management: • Users have the ability to change their passwords securely within the app. • Password change process adds an extra layer of security and control for users.
Bug Fixes and Improvements: • Enhanced validation processes for a smoother user experience. • Improved handling of appointment rescheduling, ensuring accurate updates in real-time. • Streamlined service management for lawyers, making it easier to add new services.
Security: • Security measures implemented to protect user data, ensuring a safe and secure environment.
General Enhancements: • Overall app performance improvements for a more responsive and efficient experience.
